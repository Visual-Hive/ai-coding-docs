## Convention

Every route file MUST have a leading JSDoc comment with a one-line description:

```typescript
/**
 * Member dashboard — shows registered events, course progress, account settings.
 */
export const load = async ({ locals }) => { ... };
```

The generator picks up the first line of the JSDoc as the description. Routes without a JSDoc appear in the table with `[no description — add JSDoc]` so they can't hide.
